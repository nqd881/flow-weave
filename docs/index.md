# Documentation Index

`flow-weave` has one runtime model and two authoring styles.

- Use `flow-weave` for the main app/runtime entry.
- Use `flow-weave/builder` for advanced fluent-builder APIs.
- Use `flow-weave/decorator` for core flow decorators.
- Use `flow-weave/saga` for saga plugin, saga runtime/types, and saga decorators.
